# AEMET Weather Visualizer

This is a Python app that shows weather data from accessible meteorological stations in the AEMET OpenData database. It displays temperature, humidity, wind speed, and wind direction in charts. You can select a station and view the latest 12 hours of data.

## How to Use

1. Run the 'Weather Visualizer.exe' file .
2. Select a weather station from the list.
3. Click "Select" to see the data:
   - Temperature over time
   - Humidity over time
   - Wind speed over time
   - Wind direction (rose chart)

## About the Code

- 'main.py': The main program with the window and graphs.
- 'Functions.py': Helper functions to get data from AEMET.
- The .exe file lets you run it without installing Python.

## Library Requirements

- pandas
- numpy
- matplotlib
- Tkinter
- requests
